# 组件间的通信方式

呈现页面的开发就是组件的开发，那组件与组件，组件与模块，组件与页面之间是如何通信呢？

组件的通信采用的是事件订阅/发布模式来处理的。定义一个全局的事件管理中心，事件的订阅，发布，销毁统一管理。


### 实现步骤

#### 第一步：实例化一个空的regular构造函数。

```
import RegularJS from 'regularjs';

let EvtController = RegularJS.extend({});

let evtController = new EvtController();

export default evtController;
```

#### 第二步：在这个实例化的对象上做事件的订阅，发布，销毁操作。

regularjs内已经对事件的操作方法做好了封装，所以我们直接用就行了。

事件订阅：

```
evtController.$on('updateCartNum', this.updateCartNum.bind(this));
```


事件触发：

```
evtController.$emit('updateCartNum', cbdata.cartGoodsNum);
```

事件销毁：

```
evtController.$off('firstScreenRegionLoaded');
```

之前采用的方式是，将回调函数作为参数传入进行调用。过程比较复杂，而且不科学，改成现在这种方式会好管理些。

#### regularJs 事件定义源码

```
var API = {
	  $on: function(event, fn, desc) {
	    if(typeof event === "object" && event){
	      for (var i in event) {
	        this.$on(i, event[i], fn);
	      }
	    }else{
	      desc = desc || {};
	      // @patch: for list
	      var context = this;
	      event = fix(event);
	      var handles = context._handles || (context._handles = {}),
	        calls = handles[event] || (handles[event] = []);
	      var realFn;
	      if(desc.once){
	        realFn = function(){
	          fn.apply( this, arguments )
	          this.$off(event, fn);
	        }
	        // @FIX: if  same fn
	        fn.real = realFn;
	      }
	      calls.push( realFn || fn );
	    }
	    return this;
	  },
	  $off: function(event, fn) {
	    var context = this;
	    if(!context._handles) return;
	    if(!event) this._handles = {};
	    var handles = context._handles,
	      calls;

	    event = fix(event);
	    if (calls = handles[event]) {
	      if (!fn) {
	        handles[event] = [];
	        return context;
	      }
	      fn = fn.real || fn;
	      for (var i = 0, len = calls.length; i < len; i++) {
	        if (fn === calls[i]) {
	          calls.splice(i, 1);
	          return context;
	        }
	      }
	    }
	    return context;
	  },
	  // bubble event
	  $emit: function(event){
	    // @patch: for list
	    var context = this;
	    var handles = context._handles, calls, args, type;
	    if(!event) return;
	    var args = _.slice(arguments, 1);
	    var type = fix(event);

	    if(!handles) return context;
	    if (!(calls = handles[type])) return context;

	    if(calls.length > 1){ // handle, when first is off the event
	      calls = calls.slice();
	    }
	    
	    for (var i = 0, len = calls.length; i < len; i++) {
	      if(typeof calls[i] === 'function') calls[i].apply(context, args)
	    }
	    return context;
	  },
	  // capture  event
	  $once: function(event, fn){
	    var args = _.slice(arguments);
	    args.push({once: true})
	    return this.$on.apply(this, args);
	  }
	}
```
